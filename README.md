## 📌 Project Overview

The **Bank Deposit Subscription Analysis Dashboard** is an interactive Power BI project designed to analyze customer subscription behavior for term deposits. The dataset was downloaded from **Kaggle**, cleaned using **Microsoft Excel**, and transformed into an insightful dashboard in **Power BI**.

The dashboard helps identify customer trends, subscription performance, campaign effectiveness, and demographic insights that support data-driven business decisions.

## 📂 Dataset Information

- Job
- Marital Status
- Education
- Account Balance
- Housing Loan
- Personal Loan
- Contact Month
- Campaign Details
- Deposit Subscription (Yes / No)

## 📌 Key Insights

- 📈 Total Customers: **11.16K**
- ✅ Total Subscribed Customers: **5,289**
- 📊 Conversion Rate: **47%**
- 💰 Average Account Balance: **1.53K**
- 👨‍💼 Blue-collar and management occupations contribute a large share of subscriptions.
- 📅 Deposit subscriptions vary significantly across different months.
- 📢 Campaign performance impacts customer subscription outcomes.
